Cinema Effects

Spotlight Blobs

Soft, blurry light blobs that fade in and out randomly to create a cinematic atmosphere.

  • Filter: blur(60px)
  • Transition: 1.5s ease-in-out
  • Colors: Warm cream/amber gradient
  • Behavior: Random position, random size, fade in/out

Ambient Glow

Subtle warm glow applied where light elements meet dark backgrounds.

  • Type: Radial gradient
  • Color: rgba(255, 200, 150, 0.08)
  • Purpose: Creates depth and warmth

Key Screen Layouts

Welcome / Intro Screen

  • Background: #1a1a18 (Dark)
  • Title Position: Top-left, 32px left padding, 60px top padding
  • Reel: 780px, positioned right edge (half visible), centered vertically at 43%
  • Button: Fixed 48px from bottom, 24px horizontal padding
  • Terms Text: Centered below button, 12px size, muted color
  • Effects: Random spotlight blobs, ambient glow on reel edge

Auth Screen

  • Background: Looping video with 70% black overlay
  • Video Transition: Crossfade loop (800ms fade, trigger 1s before end)
  • Logo: 56x56px app icon, centered above title
  • Title: 24px, centered ("Welcome!" / "Enter Verification Code")
  • Subtitle: 16px, muted gray, centered
  • Input Container: Country selector + phone input, 12px border radius
  • Button: Full width, 18px padding, 12px border radius
  • Terms: 13px, centered, orange links

Onboarding Flow

  • Background: Same video background as auth
  • Steps: Name â†’ (Optional: Style) â†’ Voice Recording
  • Icon Container: 50px padding, orange tinted background, Sparkles or Mic icon
  • Step Titles: "Welcome to Reelful" / "Your Style" / "Record Your Voice"
  • Back Button: Top-left, 28px ChevronLeft, appears on step 2+
  • Voice Note: Hint card with "Don't want to record? No worries!" message
  • Skip Button: Text link, underlined, "Skip for now"

Feed / Gallery Screen

  • Header: "Reelful" title + icon, Credits badge, Settings gear
  • Grid: 3-column, 8px spacing, 1.5:1 aspect ratio thumbnails
  • Thumbnail Overlay: Bottom-left label, 75% black background, 6px radius
  • FAB: 64x64px, 16px border radius, bottom-right, 24px from edge
  • Empty State: "Start a project!" + "Start Creating" button
  • Long Press: Context menu appears next to thumbnail
  • Pull to Refresh: Orange refresh indicator

Composer Screen

  • Header: X close button left, forward arrow right (for drafts with scripts)
  • Title: "Hey, [Name], share your story!" - 24px, centered
  • Media Section: "Upload Media (optimal: 5-6 files)" label
  • Media Grid: Horizontal scroll, 80x80px thumbnails, X remove button
  • Add Button: Camera icon + "Add More Media", orange background
  • iCloud Note: "â˜ï¸ Videos stored in iCloud may take a moment to load"
  • Description Input: Multiline, 100px min height, placeholder text
  • Example Card: Orange-tinted hint with sample story
  • Generate Button: "Generate Script", full width, orange

Settings Modal

  • Presentation: Bottom sheet, 94% height, slide-up animation
  • Drag Handle: 40x5px, #333 color, 3px radius, centered
  • Close Button: X icon, top-left, 32x32 touch area
  • Profile Section: Name display, 24px, centered
  • Menu Items: Icon + text + chevron, 16px vertical padding
  • Menu Items: Pro/Credits, Account, Voice Settings, Voice Clone, About
  • Logout: Red text, centered
  • Delete Account: Fixed at bottom, red text + trash icon
  • Sub-modals: 85% width, centered, 16px border radius
  • Gesture: Swipe down to close, spring back if partial
